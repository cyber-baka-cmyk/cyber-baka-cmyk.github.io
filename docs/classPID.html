Detailed Description

ПИД‑регулятор в инкрементной форме.

Реализует u(k) = u(k-1) + Δu, Δu = q0*e(k) + q1*e(k-1) + q2*e(k-2), где: q0 = K*(1 + TD/T0), q1 = -K*(1 + 2*TD/T0 - T0/T), q2 = K*(TD/T0).
